## Matriu RACI

| Tasca               |    Pau    |     Vicenç    |
| :------------------ | :-------: | :-----------: |
| **T01** Anàlisi     |   **R**   | **A · C · I** |
| **T02** Web         |   **R**   |   **A · C**   |
| **T03** Fitxers     | **A · C** |     **R**     |
| **T04** Impressores | **A · I** |     **R**     |
| **T05** Vídeos      |   **R**   |   **A · C**   |
| **T06** Legal       | **A · C** |     **R**     |
| **T07** Núvol       | **R · A** |   **C · I**   |
| **T08** Web final   |   **A**   |   **R · C**   |

***

### Llegenda RACI

*   **R** Responsable: fa la feina principal
*   **A** Validador: revisa i dona el vistiplau final
*   **C** Consultat: ajuda, dona opinió o revisa durant el procés
*   **I** Informat: està al corrent, però no participa directament
